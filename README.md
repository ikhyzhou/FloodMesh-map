Cesium 三维数据可视化练习项目

一个基于Cesium.js的简单三维地理数据展示页面，实现了基本的图层切换和时间轴功能。

在线页面：https://ikhyzhou.github.io/FloodMesh-map/

项目说明

这是我在学习WebGIS开发过程中做的一个练习项目，主要目的是：
1. 学习Cesium.js三维地球的使用
2. 实现多图层数据的管理和切换
3. 添加简单的时间轴播放控制

功能特点

- 加载并显示10个图层的地理数据
- 通过输入框或按钮切换不同图层
- 简单的时间轴播放控制（播放/暂停/上一帧/下一帧）
- 响应式UI界面

技术栈

- Cesium.js
- 原生JavaScript
- HTML5/CSS3

使用方式

1. 打开页面后等待数据加载（可能会比较慢）
2. 输入1-10的数字选择图层
3. 点击显示按钮或按Enter键
4. 使用时间轴控件播放时序数据


遇到的挑战和解决方案

1. 数据加载慢：文件较大，加载需要时间
2. 图层切换：通过控制Cesium Primitive的show属性实现
3. 异步处理：用async/await管理数据加载流程
4. 数据污染：避免全局变量，使用局部作用域

学习收获

通过这个项目，我学会了：
- Cesium.js的基本使用和API调用
- 三维地理数据的处理和显示
- 前端异步编程的实际应用
- 简单的UI交互设计

备注

- 数据文件较大，首次加载可能需要几十秒
- 由于是练习项目，代码结构可能不够优化
- 功能相对基础，适合学习和演示使用
